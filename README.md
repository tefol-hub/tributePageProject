# Tribute Page – Dr. Milton Friedman

This project is a **responsive tribute page** built as part of the
[freeCodeCamp Responsive Web Design Certification](https://www.freecodecamp.org/).

It presents a structured biography of **Dr. Milton Friedman**, focusing on clean layout, typography, and semantic HTML structure.

---

## 🚀 Live Demo

👉 https://tefol-hub.github.io/tribute-page/

---

## 💼 Project Highlights

* Built a **fully responsive single-page layout**
* Used **semantic HTML elements** for accessibility and structure:

  * `<main>`, `<figure>`, `<figcaption>`, `<ul>`
* Designed a clean **content hierarchy (title → image → timeline → biography)**
* Implemented **image styling with filters and responsive scaling**
* Applied **modern UI styling (shadows, spacing, centered layout)**
* Created a structured **timeline-based biography section**
* Included an external reference link for extended reading

---

## 🛠️ Tech Stack

* HTML5 (semantic structure, content organization)
* CSS3 (layout, spacing, typography, image styling)
